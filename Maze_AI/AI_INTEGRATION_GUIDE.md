# 迷宫游戏强化学习集成指南

## 概述

已将强化学习（Q-Learning）算法成功集成到迷宫游戏中。系统包含以下文件：

### 新增文件

1. **maze_env.py** - 迷宫环境类
   - 基于 `mapp1.py` 中的6×6迷宫地图
   - 实现状态转移、奖励计算等环境逻辑
   - 支持4个基本动作：上、下、左、右

2. **q_agent.py** - Q-Learning智能体
   - 实现标准的Q-Learning算法
   - 使用ε-greedy策略进行探索-利用权衡
   - 包含学习更新和动作选择逻辑

3. **main_ai.py** - 集成了强化学习的改进版游戏
   - 支持两种游戏模式：
     - **AI自动演示模式**：AI自动找到迷宫最优路径
     - **手动操作模式**：用户手动控制角色

## 使用方法

### 运行游戏

```bash
cd e:\Project\tank-battle\Maze_AI
python main_ai.py
```

### 游戏流程

1. **自动训练阶段**
   - 程序启动时会自动训练 AI（500轮次）
   - 显示训练进度条
   - 训练完成后进入菜单

2. **菜单选项**
   - 按 `1` - AI自动演示模式
   - 按 `2` - 手动操作模式
   - 按 `Q` - 退出游戏

3. **AI自动演示模式**
   - AI根据学习到的策略自动移动
   - 显示步数和移动过程
   - 到达目标后显示"AI Win!"
   - 按任意键返回菜单

4. **手动操作模式**
   - 使用方向键 ↑↓←→ 移动角色
   - 按 `ESC` 返回菜单
   - 到达红色目标点时赢得游戏
   - 显示完成需要的步数

## 技术细节

### 状态表示
- 状态 = 房间X坐标 × 网格宽度 + 房间Y坐标
- 总共36个状态（6×6网格）

### 动作映射
- 0: 向上移动
- 1: 向下移动 
- 2: 向左移动
- 3: 向右移动

### 奖励设计
- 到达目标：+100
- 撞墙：-1.0
- 每步扣分：-0.1（鼓励最短路径）

### Q-Learning参数
- 学习率 (α): 0.1
- 折扣因子 (γ): 0.9
- 初始探索率 (ε): 1.0
- 探索率衰减: 0.995
- 最小探索率: 0.1
