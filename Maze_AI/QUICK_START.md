## 参数优化

如果想调整AI性能，编辑 `main_ai.py` 中的参数：

```python
TRAINING_EPISODES = 500      # 改大数字 → 训练更彻底
```

或编辑 `q_agent.py` 中的超参数：

```python
self.alpha = 0.1             # 学习率：越大学得越快（但不稳定）
self.gamma = 0.9             # 折扣因子：越大越看重未来奖励
self.epsilon_decay = 0.995   # 衰减速度：越小越快减少探索
```

## 常见问题

**Q: AI为什么不能直接赢？**
A: 这是正常的。600轮训练后，AI应该能找到接近最优路径。如果要更好的表现，增加 TRAINING_EPISODES。

**Q: 如何修改迷宫？**
A: 编辑 `mapp1.py` 中的 `map_list`，其中：
- `1` = 墙
- `0` = 通路
- `3` = 起点（会被替换为通路）

**Q: 可以用其他强化学习算法吗？**
A: 可以！参考 `ai-maze-navigation/` 中的代码，可以实现 DQN、Policy Gradient 等。
