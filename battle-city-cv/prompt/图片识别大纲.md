---

## 🎯 核心思路

---

## ✅ 你要做的事情

### ✔ 1. 面积过滤

```python
if 200 < area < 2000
```

👉 但要分类调整：

| 目标 | 面积 |
| -- | -- |
| 坦克 | 中等 |
| 子弹 | 很小 |
| 墙  | 很大 |

---

### ✔ 2. 宽高比过滤（关键！）

```python
ratio = w / h
```

| 目标 | 特点    |
| -- | ----- |
| 坦克 | 接近正方形 |
| 砖墙 | 长条    |
| 子弹 | 小点    |

---

### ✔ 3. 轮廓复杂度

```python
cv2.approxPolyDP
```

👉 坦克形状复杂
👉 墙 = 矩形
